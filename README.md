# Library Acquisition System

## Project Overview
The Library Acquisition System is designed to efficiently manage the acquisition of library resources, including books, journals, and other materials. The project focuses on enhancing the user experience and administrative efficiency in libraries.

## Technologies Used
- **PHP**: 26.5% - Server-side scripting language used for backend.
- **Blade**: 55.2% - Templating engine for dynamic page generation.
- **HTML**: 17.8% - Markup language for structuring web pages.

## Features
- User-friendly interface for library staff and patrons.
- Efficient tracking of acquisitions and inventory.
- Detailed reporting and analytics on library resources.

## Installation and Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/tyser1995/library_aquisition_system.git
   ```
2. Install dependencies:
   ```bash
   composer install
   ```
3. Set up the database and environment variables.

## Usage
- Access the application via `http://localhost:8000` after starting the server.
